# Rust Variables

### Prerequisites
##### Installing Rust
- Install Rust
  ```yaml
  curl --proto '=https' --tlsv1.2 https://sh.rustup.rs -sSf | sh
  ```
- Check version
  ```yaml
  rustc --version
  ```
- Update version
  ```yaml
  rustup update
  ```

### Getting Started
- Clone the repository
  ```yaml
  git clone https://github.com/Raad05/rust-variables.git
  ```
- Build the program
  ```yaml
  cargo build
  ```
- Run the program
  ```yaml
  cargo run
  ```
